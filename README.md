# ИИ-агент-тьютор

## Функционал

1. Организационная помощь — отвечает на вопросы о расписании, дедлайнах, статусах проверок, способах сдачи заданий.

2. Учебная поддержка — даёт советы и подсказки, не выполняя задания вместо слушателя.

3. Углубление знаний — отвечает на содержательные вопросы по программным модулям курса.

## Стек

- Чат-интерфейс: Telegram Bot API

- Орекестрация: n8n self-host

- LLM: OLlama/Anthropic

- БД: PostgreSQL
